# Data Availability

Processed data, code, and reproducible analysis scripts will be deposited in this GitHub repository and archived through Zenodo upon publication.

Raw Google Street View imagery is subject to third-party licensing restrictions and will not be redistributed publicly. Derived metrics and image metadata will be provided, and additional research materials may be made available from the corresponding author upon reasonable request.

## Planned Public Release

The planned public release will include:

- Processed node-level tabular metrics required to reproduce the reported models.
- Hexagon-level sampling and cluster metadata.
- Model input dictionaries and variable definitions.
- Reproducible scripts for data integration, statistical diagnostics, model benchmarking, and figure/table regeneration.
- Documentation describing how raw imagery was collected and transformed into derived semantic metrics.

## Restricted Materials

The following materials are not included in this pre-publication repository:

- Raw Google Street View image files.
- API keys, authentication tokens, local credentials, and Earth Engine project configuration.
- Local cache files, logs, and intermediate processing artifacts.
- Submitted manuscript source files, journal PDFs, and full-resolution production figures during peer review.
